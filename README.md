# Store Management System (C++)

This project is a simple store management system developed in C++.
It is designed to manage products, inventory, and basic store operations.

## Features
- Add new product
- Display all products
- Calculate inventory value
- Display low stock products
- Search product by name
- Save data to a text file
- Load data from a text file
- Sort products by price (ascending)

## Technical Details
- Language: C++
- Data storage: Array and text file
- Maximum number of products: 50

## Usage
Run the program and use the menu options to manage store products.
